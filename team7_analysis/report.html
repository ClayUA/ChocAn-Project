Paragraph

The hardware end of the ChocAn system will consist of terminals placed around the facility, with

physical ID cards handed out to members. All other aspects of this project consist of software

implementation. The provider and member data will be stored in the system alongside a comprehensive

database of services and their information. The system will interact with Acme Accounting Services

concerning the ETF records, and ChocAn’s larger systems to send and receive updated

information/billings. Individual users will interact with the system via the terminals, with members’ ID

cards serving a role in Healthcare Providers being able to verify membership of their current client. The

terminals’ interfaces will allow for navigation between each of the uses.
