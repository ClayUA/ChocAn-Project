Use Case: Request Software Product
Context: Provider requests a copy of the Provider Directory
Actors: ChocAn Provider
Main Success Scenario:
	1. Provider opens terminal
	2. Provider navigates to Provider Directory
	3. Provider fills out form to receiver an email copy of directory
	4. System sends information to ChocAn Data Center
	5. Provider receives email with Directory attached

Use Case: Add Provider
Context: Operator can add a new Provider to the database
Actors: ChocAn Operator
Main Success Scenario:
	1. Operator opens terminal
	2. Operator navigates to "Add Provider" menu
	3. Operator fills out form with Provider's info
	4. That info is added to database
